#  Notes :

#    - folium génère la carte Leaflet.js en HTML pur (pas de serveur web requis).
#    - networkx fournit les algorithmes de graphe (Dijkstra, A*).
#    - osmnx est optionnel ici (on parse le fichier .osm directement avec xml.etree),
#      mais il est inclus pour permettre de télécharger d'autres zones OSM à l'avenir.

# --- Interface Graphique ---
PyQt6>=6.6.0
PyQt6-WebEngine>=6.6.0      # Nécessaire pour QWebEngineView (carte interactive)

# --- Cartographie ---
folium>=0.15.0              # Génération de cartes Leaflet.js (HTML interactif)

# --- Algorithmes de graphe (pathfinding) ---

# --- Données géospatiales (optionnel — pour extensions futures) ---

pyproj>=3.6.0               # Projection de coordonnées GPS (AEQD locale pour export JSON)
